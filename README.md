# Eve Deshon — Portfolio (starter)

This is a lightweight, dependency-free portfolio site inspired by the simple grid layout of `macallisterstatham.com`.

## Run locally

From the `eve-portfolio/` folder, run:

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

## Edit content

- Home page: `index.html`
- Case studies: `case-studies/*.html`
- Styles: `styles.css`
