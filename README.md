# Eve Deshon — Portfolio (Next.js)

Next.js portfolio with Work, Experience (animated timeline), and Contact.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

- **Home:** /
- **Work:** /work
- **Experience:** /experience
- **Contact:** /contact
- **Case studies:** /case-studies/monk.html (and others in `public/case-studies/`)

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy to [Vercel](https://vercel.com) — set **Root Directory** to the repo root (this is the Next.js app).
